
### EV Adoption in India (2019‚Äì2024)

This project analyzes the **growth of Electric Vehicle (EV) adoption across major Indian states from 2019 to 2024**.
It identifies trends, top-performing states, and category-wise (2W, 3W, 4W, Bus) adoption patterns using Python-based data analysis.

---

###  Objectives

* Analyze **yearly growth** of EV adoption in India
* Identify **top states** contributing to total EV sales
* Compare **category-wise trends** (2W, 3W, 4W, Buses)
* Visualize insights using **Python (Matplotlib, Seaborn)**

---

###  Tech Stack

* **Python**, **Pandas**, **Matplotlib**, **Seaborn**
* **Google Colab** for data exploration
* **Power BI** (optional) for dashboard visualization

---

###  Files

| File                               | Description                                                |
| ---------------------------------- | ---------------------------------------------------------- |
| `EV_Adoption_India_Analysis.ipynb` | Main Jupyter notebook with all analysis and visualizations |
| `ev_adoption_india.csv`            | Dataset of EV sales by state, year, and category           |

---

###  Key Insights

* India‚Äôs EV sales show a **steady rise from 2019 to 2024**
* **2-wheelers and 3-wheelers** dominate the market share
* **Maharashtra, Delhi, and Karnataka** lead in total EV adoption
* Northern and central states show strong late-stage growth momentum
